## 🚀 Quick Action Guide

### "I want to preview locally"
```
1. Open QUICK_START.md
2. Use python -m http.server 8000
3. Visit http://localhost:8000
```

### "I want to customize my name/email"
```
1. Open index.html
2. Find your name (search: "Mohammad Ayan Husain")
3. Replace with your name
4. Find your email (search: "your.email@example.com")
5. Replace with your email
6. Save and refresh
```

### "I want to change colors"
```
1. Open CUSTOMIZATION.md
2. Find Color Schemes section
3. Copy color values
4. Open css/style.css
5. Paste into --primary-color and --secondary-color
6. Refresh browser
```

### "I want to deploy on GitHub Pages"
```
1. Open DEPLOYMENT_GUIDE.md
2. Go to "Option 1: GitHub Pages"
3. Follow steps 1-5
4. Done! Your site is live
```

### "I want to deploy on Netlify"
```
1. Open DEPLOYMENT_GUIDE.md
2. Go to "Option 2: Netlify"
3. Follow steps 1-3
4. Done! Your site is live
```

### "I want to deploy on Vercel"
```
1. Open DEPLOYMENT_GUIDE.md
2. Go to "Option 3: Vercel"
3. Follow steps 1-3
4. Done! Your site is live
```

### "I want to add more projects"
```
1. Open index.html
2. Find the projects section
3. Copy a <article class="project-card"> block
4. Paste it below
5. Update title, description, links
6. Save and refresh
```

### "I want to add my resume"
```
1. Save your resume as resume.pdf
2. Place in: assets/documents/resume.pdf
3. Done! Download button will work
```

### "I want to add my photo"
```
1. Save your photo (JPG or PNG)
2. Place in: assets/images/photo.jpg
3. Update HTML to use the image
4. (See CUSTOMIZATION.md for details)
```
